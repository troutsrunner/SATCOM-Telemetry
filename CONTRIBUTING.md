# Contributing to SATCOM-Telemetry

Thanks for your interest in contributing! This project is open source and welcomes contributions from anyone.

## How to contribute

1. Open an issue first if you want to propose a new feature or report a bug.
2. Fork the repository and create a branch for your change.
3. Make your changes in a clean, focused branch.
4. Submit a pull request with a clear title and description.

## Code style

- Follow existing code patterns in the project.
- Keep changes small and easy to review.
- Add comments or documentation for non-obvious logic.

## Testing

- If you add functionality, include tests when possible.
- Make sure existing behavior still works.

## Pull request expectations

- Describe the problem and your solution clearly.
- Reference related issues when applicable.
- Include screenshots or examples if the change affects UI or behavior.
